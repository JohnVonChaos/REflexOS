# ReflexOmega_TheWORD

## Recent Updates

- Background cognition now runs as a chained 3-layer cycle (Subconscious → Conscious → Synthesis) during idle time by default. This mirrors the active pipeline to make background processing more predictable and traceable.
- Added per-workflow stage setting `backgroundRunMode` (choices: `chained` | `independent`) accessible in the **Workflow Designer**. `chained` runs the full 3-layer chain; `independent` runs synthesis-only for lightweight periodic tasks.
